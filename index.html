// What is Prompt Engineering?

Learn to write prompts that work

Prompt engineering is about crafting clear, logical, and adaptive instructions that guide AI systems to produce useful results. Start by reading the short overview below, then move into hands-on practice with real prompt tasks.

How you learn here

  • Read a task scenario and goal.
  • Analyze key requirements and organize your ideas.
  • Write your own prompt in the box below.
  • Submit it to get a score and learn how to improve.

The CLEAR scoring framework

Your prompt is evaluated across five dimensions: Concise, Logical, Explicit, Adaptive, and Reflective. Each dimension is scored 0–4, for a total of 20 points.

Prompt task example

Task: Design a prompt that asks an AI to write a short persuasive email encouraging a friend to try a new productivity app.

Your prompt should be clear about the tone, target audience, and required structure (e.g., greeting, benefits list, call to action).

// Platform Overview

What is MetaForge?

MetaForge is a structured educational platform designed to develop your prompt engineering skills through deliberate, scaffolded practice with AI systems.

Prompt engineering is rapidly becoming a critical skill in the AI era. MetaForge bridges the gap between theoretical frameworks and hands-on practice by providing structured challenges that progressively build your ability to communicate complex instructions to large language models.

Built on Bloom's Revised Taxonomy, the platform ensures you develop a complete cognitive toolkit — from foundational recall and comprehension all the way to synthesis and creative generation of novel prompt structures.
